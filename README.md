## What is the difference between var, let, and const?
var: is function-scoped and can be redeclared and reassigned. It is also hoisted to the top of its scope.
let: is block-scoped and cannot be redeclared in the same scope, but it can be reassigned.
const: is block-scoped and cannot be redeclared or reassigned after it is defined.

## What is the spread operator (...)?
The spread operator is used to expand elements of an iterable such as an array or object. It helps copy, merge, or pass elements easily in JavaScript.

## What is the difference between map(), filter(), and forEach()?
map(): creates a new array by transforming every element of an existing array.

filter(): creates a new array containing only the elements that match a specific condition.
forEach(): loops through an array and performs an action for each element but does not return a new array.

## What is an arrow function?
An arrow function is a shorter syntax for writing functions in JavaScript. It makes code cleaner and more concise and handles the this keyword differently compared to regular functions.

## What are template literals?
Template literals are a modern way to work with strings in JavaScript. They allow embedding variables and expressions directly inside a string and also support multi-line strings. They are written using backticks instead of quotation marks.
